# Sobre

Refatoração de um site implementando recursos de acessibilidade no HTML, CSS (Stylesheet) e JS.

## Recursos de Acessibilidade

- Atributos ARIA;
- Alt (texto alternativo);
- Tab-index;
- Menu de acessibilidade.

## Tecnologias utilizadas

- Bootstrap;
- ScrollRevealjs;
- HTML;
- CSS;
- JS.
